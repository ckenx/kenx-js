# Kenx Documentation

Welcome to Kenx-JS, a **config-first development framework** for building Node.js, Deno, and Bun applications with speed and simplicity.

## What is Kenx?

Kenx aims to revolutionize web application development by focusing on three core principles:

1. **Simple and seamless setup** - Get started in minutes with minimal boilerplate
2. **Useful development assistance** - Built-in tools and automation that enhance productivity
3. **Easy maintenance** - Clean architecture and clear patterns for long-term sustainability

Based on years of experience from developers across different backgrounds, Kenx provides a unified approach that speeds up development while maintaining code quality. Do less work, focus on what matters, and build for the future.
